Git

  • "GitHub Desktop" - программа для работы.
  • "username.github.io" - адрес своего сайта.
  • git clone https://github.com/username/samp - скачивание репозитория.
  • git status - статус.
  • git add . - добавить всё.
  • git commit -m "v0.1" - название версии исправления.
  • git push - залить на GitHub.
  • git add -A folder - добавить папку.
  • git add file.txt - добавить файл.
  • Создать файл:
  • ".gitignore"
  • Содержимое файла:
  • "worlds/" - папки.
  • "server.log" - файлы.
  • git config --global user.email "mail@ya.ru" - указать почту.
  • git config --global user.name "username" - указать пользователя.
  • git config --list - показывает настройки конфигурации.
